# Security policy / Política de seguridad

## Scope / Alcance

This repository is **documentation**: checklists, prompts, templates, rules, and
examples for static review of third-party skills. It is not a scanner, not a
service, and runs no code.

Este repositorio es **documentación**: checklists, prompts, plantillas, reglas y
ejemplos para la revisión estática de skills de terceros. No es un scanner, no es
un servicio y no ejecuta código.

---

## Reporting an issue with this repo / Reportar un problema de este repo

If you find a problem in **this repository's own content** — for example, an
example fragment that is too operational, a mistake that could mislead a
reviewer, or broken/misleading instructions — please open an issue or contact
the maintainers describing the file and the concern.

Si encontrás un problema en **el contenido de este repositorio** — por ejemplo,
un fragmento de ejemplo demasiado operativo, un error que podría confundir a
quien revisa, o instrucciones rotas o engañosas — abrí un issue o contactá a los
mantenedores describiendo el archivo y la preocupación.

**Do not include functional payloads in reports.** Describe the problem; do not
paste working malware, real payloads, or reassemblable attack code.

**No incluyas payloads funcionales en los reportes.** Describí el problema; no
pegues malware funcional, payloads reales ni código de ataque rearmable.

---

## What this project does NOT do / Lo que este proyecto NO hace

- **It does not maintain a list of bad actors.** We do not track, name, or
  accuse projects, authors, or companies.
- **It does not handle reports about real third-party skills.** If you believe
  you found real malware in a third-party skill, report it to the relevant
  **platform or source** (where the skill is distributed), not here.
- **It does not certify or guarantee safety.** SAFE is the result of a static
  review process, never a guarantee.

- **No mantiene una lista de malos actores.** No rastreamos, nombramos ni
  acusamos a proyectos, autores ni empresas.
- **No gestiona reportes sobre skills reales de terceros.** Si creés que
  encontraste malware real en una skill de terceros, reportalo a la
  **plataforma u origen** correspondiente (donde se distribuye la skill), no acá.
- **No certifica ni garantiza seguridad.** SAFE es el resultado de un proceso de
  revisión estática, nunca una garantía.

---

## Responsible handling / Manejo responsable

If your review of a real skill turns up something dangerous, follow the
[anti-accusation policy](README.md#anti-accusation-policy): document with
evidence, report to the platform/source, and avoid public accusations against
people.

Si tu revisión de una skill real encuentra algo peligroso, seguí la
[política anti-acusación](README.es.md#política-anti-acusación): documentá con
evidencia, reportá a la plataforma/origen y evitá las acusaciones públicas
contra personas.
